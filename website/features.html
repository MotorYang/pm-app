功能详解

Pomo 整合了开发者日常工作流程中最常用的功能模块， 让你在一个应用中完成项目管理、文档编辑、版本控制和凭据存储。

项目与工作流

高效管理本地项目，快速切换工作上下文

项目仪表盘

一览所有项目状态，包括最近活动、Git 状态和待办事项

快速项目切换

一键切换不同项目工作区，保持上下文和窗口状态

快捷方式管理

为常用操作创建快捷方式，支持自定义命令和脚本

终端集成

内置终端快速访问，支持在项目目录直接打开系统终端

文件管理器联动

一键在系统文件管理器中打开项目目录

项目分组与标签

使用分组和标签组织项目，快速筛选和定位

DocVault 文档库

项目文档的集中管理，支持 Markdown 和 PDF

Markdown 编辑器

功能完整的 Markdown 编辑器，支持实时预览和语法高亮

PDF 阅读器

内置 PDF 查看器，无需切换应用即可阅读技术文档

文档树结构

使用树形结构组织文档，支持无限层级和拖拽排序

最近文档

快速访问最近编辑或查看的文档

全文搜索

在所有文档中快速搜索关键词和内容

本地存储

所有文档存储在本地，无云端同步，数据完全掌控

Git 版本控制

可视化 Git 操作，告别繁琐的命令行

可视化状态

直观显示文件变更状态：新增、修改、删除、未跟踪

提交历史

浏览完整的提交历史，查看每次提交的详细信息和差异

分支管理

创建、切换、合并分支，可视化分支关系图

暂存/取消暂存

点击即可暂存或取消暂存文件，支持部分文件暂存

提交界面

简洁的提交界面，支持多行提交信息和快捷键操作

差异对比

并排或内联显示文件差异，清晰了解每处变更

安全保险库

军用级加密保护你的敏感数据

AES-256-GCM 加密

使用行业标准的 AES-256-GCM 算法加密所有存储数据

Argon2 密码派生

使用 Argon2 算法从主密码派生加密密钥，抵御暴力破解

文件名混淆

加密文件的文件名和路径信息，防止元数据泄露

完全本地存储

加密数据仅存储在本地，不经过任何服务器

密钥管理

安全存储 API 密钥、数据库密码、令牌等敏感凭据

敏感文件存储

加密存储证书、配置文件等敏感文件

用户体验

为开发者精心设计的界面和交互

卡通风格 UI

友好的视觉设计，在专业中带有一丝趣味

亮色/暗色主题

支持亮色和暗色两种主题，适应不同的工作环境

紧凑布局

高信息密度设计，最大化利用屏幕空间

响应式调整

窗口大小变化时自动调整布局，保持良好的使用体验

键盘快捷键

丰富的键盘快捷键支持，提高操作效率

高性能

基于 Tauri 和 Rust 构建，启动快速，内存占用低
