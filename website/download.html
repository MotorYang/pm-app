系统要求

Pomo 基于 Tauri 构建，资源占用极低，可在大多数现代计算机上流畅运行。

Windows

  • Windows 10 或更高版本
  • x64 架构处理器
  • WebView2 运行时（Windows 11 已预装）
  • 至少 100MB 可用磁盘空间

macOS

  • macOS 11 Big Sur 或更高版本
  • 支持 Intel 和 Apple Silicon
  • 至少 100MB 可用磁盘空间

Linux

  • 支持 Ubuntu 20.04+、Fedora 33+ 等
  • 需要 WebKitGTK 运行时
  • x64 架构处理器
  • 至少 100MB 可用磁盘空间

从源码构建

如果你更倾向于从源码构建，可以克隆仓库并按照 README 中的说明进行编译。 

# 克隆仓库
git clone https://github.com/motoryang/pm-app.git

# 进入目录
cd pm-app

# 安装依赖
pnpm install

# 开发模式运行
pnpm tauri dev

# 构建生产版本
pnpm tauri build
