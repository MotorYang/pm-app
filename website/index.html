本地优先的开发者工作空间

项目、文档、Git 和密钥 — 安全地集成在一个应用中

Pomo 帮助开发者在本地管理项目、文档、Git 仓库和加密凭据，无需依赖云端服务。 基于 Tauri 2.0 和 Vue 3 构建，为追求高效和安全的开发者打造。

为什么选择 Pomo？

开发者的日常工作涉及多个分散的工具和平台。项目笔记散落各处，密钥存储不安全， Git 操作需要频繁切换终端。Pomo 将这一切统一到一个本地应用中。

项目知识碎片化

README、API 文档、配置说明分散在不同的地方，难以快速查找和管理。

密钥管理混乱

API 密钥、数据库密码存放在 .env 文件或便签中，安全性难以保证。

Git 操作繁琐

频繁切换终端执行 Git 命令，查看状态、对比差异需要多个步骤。

工作流分散

项目管理、文档编辑、版本控制使用不同工具，上下文切换消耗精力。

核心功能

三大模块，覆盖开发者日常工作流程的核心需求

项目工作区

本地项目仪表盘，支持快捷方式、终端集成和文件管理器联动，快速切换和管理多个项目。

DocVault 文档库

内置 Markdown 编辑器和 PDF 阅读器，支持项目文档树结构组织，所有文档本地存储。

安全保险库

使用 AES-256-GCM 加密存储 API 密钥、令牌和敏感文件。Argon2 主密码保护，文件名混淆。

简洁高效的界面设计

卡通风格 UI，支持亮色/暗色主题，紧凑布局最大化工作区域
